ZFS on Linux APT Repository

I run this APT repository for my personal use.

This repository currently contains stable and experimentall ZFS packages on Linux for Ubuntu (and probably Debian, not tested).

Last updated: 14 Jan 2020. The packages in experimental repo repository were updated using latest zfs master tree and marked as version 0.8.3

For an automated script to install Ubuntu 18.04 LTS with ZFS root filesystem using stable repo packages, click here

To install:

First, add the following to your APT sources.list file:

wget -O - https://andrey42.github.com/test-workflow/apt_pub.gpg | apt-key add -

To install stable build:

sudo add-apt-repository 'deb https://andrey42.github.com/test-workflow/public bionic zfs-backports'
sudo apt update
sudo apt install --yes zfs-dkms zfsutils-linux

To install experimental build and tests:

sudo add-apt-repository 'deb https://andrey42.github.com/test-workflow/public bionic zfs-backports-experimental'
sudo apt update
sudo apt install --yes zfs-dkms zfsutils-linux zfs-test
